## Requirements

### macOS

uses Bullet stable 3.17

```
brew install bullet

```

### Linux

```
apt-get install -y -qq pkg-config
apt-get install -y -qq libbullet2.88
apt-get install -y -qq libbullet-dev=2.88+dfsg-2build1

```
